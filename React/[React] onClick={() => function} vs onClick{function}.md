## onClick={() => function} vs onClick{function} 차이

> 헷갈리는 내용

- onClick{function}는 페이지 렌더링 되자마자 바로 실행됨
- onClick={() => function}은 클릭 또는 event가 발생했을 때 콜백으로 실행됨.
